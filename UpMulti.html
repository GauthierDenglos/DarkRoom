Updates : Le multijoueur

Multijoueur (4) - 09/06/2018 -

Nous avons réussi à connecter deux ordinateurs portables en réseau local pour lancer le multijoueur. Et bonne nouvelle ça marche ! Il suffit de trouver l'adresse IP de notre Host (serveur) pour l'écrire dans celui de notre Client.

- Marile Lin @Lowbattery


Multijoueur (3) - 08/06/2018 -

La synchronisation des chronomètres dans le mode multijoueur est enfin opérationnelle, avec une particularité différente de l’autre mode de jeu (pour une seule personne) : il compte au lieu de décompter, avec une durée qui reste d’une heure. De plus, comme vous pouvez l’apercevoir sur l’image ci-dessous, des décorations ont été ajoutées avec plus d’animations !

- Marile Lin @Lowbattery


Multijoueur (2) - 01/05/2018 -

Les synchronisations des animations (avec le component Network Animation) et des déplacements (utilisation de deux points Spawn) des joueurs sont opérationnelles mais il reste cependant des légers détails à régler pour que le multi joueur soit complétement fini. Notre jeu utilisant le logiciel Unity comme plateforme, se base sur le Networking (appelé aussi Unet) de Unity. Sur le screen ci-dessus, vous pouvez remarquer les petites options à gauche qui permettront de lancer le multi joueur, sachant que le Host sera notre premier joueur et Lan Client notre deuxième joueur. Vous pouvez voir le résultat d'un lancement sur le screen ci-dessous.

- Marile Lin @Lowbattery


Multijoueur (1) 13/03/2018 -

Initialisation de la création du mode Multijoueur en réseau local de Darkroom : Notre multijoueur se limite à 2 joueurs, il consiste à avoir deux points de vue différents : le premier joueur possédera plus un point de vue « interne » car il se déplacera dans les pièces pour pouvoir récolter les indices, alors que le deuxième joueur aura plus un point de vue dit « externe », puisqu’il peut voir toutes les pièces de haut sans pour autant être vu par notre premier joueur ; il peut ainsi guider le premier joueur à trouver plus facilement les indices.

- Marile Lin @Lowbattery
